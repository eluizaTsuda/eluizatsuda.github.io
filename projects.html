Random Quote

The quote machine displays a random quote and a different background color when clicking the "New Quote" button. It also allows you to post it on Twitter by clicking the "Tweet this quote" button. The Quotes API is a simple API that returns an inspirational quote and its author. Fetch was used to get data from the API and the JSON() method to convert the response into JSON data.

Try it live and view the code on Codepen
